jQuery Rich Text Editor Plugin

A lightweight (< 500 lines of code!) rich text editor built on top of jQuery UI.

Details

This editor is designed with extensibility in mind to use as a base for integration into your own projects.

This editor is designed with the idea of progressive enhancement and is compatible with IE6+, Firefox 2+, Opera 9+, Safari 3+. Browsers which don't support the functionality will simply get a normal textarea.
